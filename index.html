DeepVoyage

I created an ocean-themed website for Activity 3, focusing on the use of tables, which turned out to be perfect for presenting a gallery layout. The moment I came across the jellyfish design, I immediately knew I wanted to build something inspired by the ocean. That single element shaped the overall concept and direction of the project.

Although I didn’t originally plan for it to turn out this way, it has become my favorite work so far. I’m aware that the project uses only basic HTML and CSS, and that some parts of the code are repetitive, but I still take pride in how it came together.
